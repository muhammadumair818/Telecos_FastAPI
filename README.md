# 🚀 Productivity AI Assistant (Experimental)

An AI-powered productivity assistant that analyzes user-provided data to generate actionable insights, forecasts trends, and recommends next steps using Machine Learning and LLMs.

---

## 📌 Project Overview

This project combines **data analytics, machine learning, and Large Language Models (LLMs)** to help users make better decisions from their data.

It allows users to:
- Upload or analyze datasets
- Visualize KPIs and trends
- Get AI-driven recommendations
- Query their own data using natural language

---

## ✨ Key Features

### 📊 1. Tracking (Data Analysis)
- Exploratory Data Analysis (EDA)
- KPI dashboards
- Correlation analysis
- Visual insights using plots

### 📈 2. Prediction (Forecasting)
- Trend analysis
- Future predictions using ML models
- Data-driven forecasting

### 🤖 3. Action Recommendations
- AI-generated insights using LLM APIs
- Context-aware suggestions
- Natural language querying of user data

---

## 🧠 AI & ML Capabilities

## 🛠️ Tech Stack

- **Backend:** FastAPI  
- **Frontend:** HTML, CSS, JavaScript  
- **Data Processing:** Pandas, NumPy  
- **Visualization:** Matplotlib, Seaborn  
- **Machine Learning:** Scikit-learn  
- **AI/LLM:** Hugging Face APIs

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/muhammadumair818/Telecos_FastAPI.git
cd Telecos_FastAPI
```
2. Install dependencies
```
pip install -r requirements.txt
```
4. Run the FastAPI server
```
uvicorn main:app --reload
```
6. Open in browser
```
http://127.0.0.1:8000
```
🐳 Run with Docker
```
docker build -t productivity-ai .
docker run -p 8000:8000 productivity-ai
```
